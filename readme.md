## Overview
The Hercules Challenge is a co-design consortium challenge available for first-year engineers through the mandatory introductory class, "Introduction to Professional Engineering." The idea behind the Hercules Challenge is to introduce humanitarian engineering to students who might otherwise never consider it.

## Pillarator

![Status](https://img.shields.io/badge/Status-WIP-2C2C2C?style=flat-square)
![MIT License](https://img.shields.io/badge/License-MIT-F3F4F4?style=flat-square&logoColor=black)
![EM](https://img.shields.io/badge/Physics-Electromagnetics-2C2C2C?style=flat-square&logo=physics&logoColor=87CEEB)

The `Pillarator` is a pill system built around an axial shake generator, meaning it relies on no external power. It has multi-compartment storage bays for pills. <!-- and an `~260mL` water bottle addon. !--> It does have a Li-Po battery to store power generated by the users natural motion while moving or shaking on-purpose. This allows it to use an e-ink display to show, via picture graphs, when each storage bay should be taken. It also has a ring LED that alerts the user during the first minute of each notification.

In humanitarian engineering, systems often help compensate for deficits. `The Pillarator` is another example of a device that does this. It addresses medication adherence by leveraging memory and consistency, two areas where machines excel. At the same time, it offers a solution for users in power-constrained environments.

The generator uses `Faraday's Law of Induction`, which states that the rate of change of flux linkage over time produces an `opposing voltage` to the direction of motion. To harness this effect, the generator stator has `10` coils with alternating winding directions. The generator's armature has `5` N52 magnets in an `N-S-S-N-` orientation. 

```
TOP (N) -> (30mm) -> core (N-S|S-N|N-S|S-N) (80mm) -> (30mm) -> BOTTOM (N)
```

The armature also has magnets acting on it, which form a system of equilibrium. This allows for small forces on the armature to cause a damped sine-wave motion rather than a single pulse. The generator generates `0.811W` at 8hz shaking frequency (`20mm` internal displacement, on-purpose shaking)

To improve the generator's output thin `0.2mm` iron laminations could be added to help direct the flux within the internal cavity. However, this addition could not be simulated due to time restrictions. It is, however, supported by:

$$V_{\text{induced}} = -\frac{d\lambda}{dz} \cdot \frac{dz}{dt} \implies -\frac{d\lambda}{dt}$$

$$\lambda(z) \propto \frac{N^2 \mu(z) A}{l} \implies \lambda(z) \propto \mu(z)$$
